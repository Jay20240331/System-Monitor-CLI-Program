# System Monitor CLI

A simple command-line tool for monitoring system performance in real time.

## Features
- Display CPU usage
- Show memory usage
- Real-time monitoring in terminal

## Tech Stack
- C#
- .NET

## How to Run
1. Clone the repository: https://github.com/Jay20240331/System-Monitor-CLI-Program

Navigate to the project folder:  cd SystemMonitorCLI

Run the program:  dotnet run

Example Output: CPU Usage: 35%   Memory Usage: 60%
